## Questions

### Question 1
**What is the time complexity of binary search on a sorted array of n elements?**

**Answer: B) O(log n)**

Binary search has logarithmic time complexity because it divides the search space in half with each comparison.

---

### Question 2
**Which data structure uses LIFO (Last In, First Out) principle?**

**Answer: B) Stack**

Stacks follow LIFO - the last element added is the first one removed.

---

### Question 3
**In object-oriented programming, what is encapsulation?**

**Answer: B) Bundling data and methods that operate on that data within a single unit**

Encapsulation combines data and methods while restricting direct access to internal state.

---

### Question 4
**What is the main purpose of SQL JOIN operations?**

**Answer: C) To combine rows from two or more tables based on a related column**

JOINs allow retrieval of data from multiple related tables.
